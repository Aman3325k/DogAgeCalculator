# DOGBREEDAGE.COM

Accurate dog age calculator by breed size.
Built with AstroJS and hosted on Cloudflare Pages.

## Tech Stack
- AstroJS (Static Site Generation)
- Tailwind CSS v4
- Vanilla JavaScript
- Cloudflare Pages (Free Hosting)

## Pages
21 pages including 12 breed-specific calculators

## Run Locally
npm install
npm run dev

## Deploy
npm run deploy
